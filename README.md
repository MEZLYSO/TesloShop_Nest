# TESLO API

1. Clone the proyect 

2. ```yarn install```

3. CLone the file ```.env.template``` rename with ```.env```

4. Change the enviroment variables

5. Up to the database 
```
docker compose up -d
```

6. Start proyect ```yarn start:dev```
